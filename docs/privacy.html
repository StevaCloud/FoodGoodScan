Politique de confidentialité

FoodGoodScan
Dernière mise à jour : 9 juillet 2026

1. Informations collectées

FoodGoodScan collecte les informations suivantes :

  • Adresse courriel — utilisée pour la création de compte et la connexion.
  • Numéro de téléphone — utilisé pour la prévention des abus d'abonnement.
  • Code postal — utilisé pour afficher les circulaires et offres de votre région.
  • Données de scan — codes-barres des produits scannés, conservés dans votre historique.
  • Localisation approximative — utilisée pour les données météo et les offres locales (non stockée de façon permanente).

2. Utilisation des données

Vos données sont utilisées exclusivement pour :

  • Fournir les fonctionnalités de l'application (scan, circulaires, listes d'épicerie, quiz)
  • Gérer votre abonnement via Stripe
  • Afficher des publicités pertinentes via AppLovin MAX (avec votre consentement)
  • Améliorer les recommandations nutritionnelles personnalisées

3. Partage des données

Nous ne vendons jamais vos données personnelles. Elles peuvent être partagées avec :

  • Stripe — traitement sécurisé des paiements
  • AppLovin — publicités dans l'application
  • Open Food Facts — base de données nutritionnelles publique et ouverte

4. Conservation des données

Vos données sont conservées tant que votre compte est actif. Vous pouvez demander la suppression complète de votre compte et de toutes vos données à tout moment en nous contactant.

5. Sécurité

Vos données sont protégées par chiffrement (HTTPS, bcrypt pour les mots de passe, JWT sécurisés). Nous appliquons des mesures techniques et organisationnelles pour protéger vos informations contre tout accès non autorisé.

6. Vos droits

Conformément à la Loi 25 (Québec) et au RGPD (Union européenne), vous avez le droit de :

  • Accéder à vos données personnelles
  • Corriger des informations inexactes
  • Demander la suppression de vos données
  • Retirer votre consentement à tout moment

7. Contact

Pour toute question concernant cette politique ou vos données :
